Universal React. Explore the source code.

## Prerequisites

* [node.js](http://nodejs.org/) Node 8+
* [yarn](https://yarnpkg.com/)

## Start Development

* `git clone https://github.com/este/este.git este`
* `cd este`
* `yarn`
* `yarn dev`

## Random Notes
